# 🎬 AI Video Assistant

An AI-powered video analysis application that transforms meetings and videos into structured insights. Upload a local video or provide a YouTube link to automatically generate transcripts, summaries, action items, key decisions, and interact with the content through an AI-powered chat interface.

---

## ✨ Features

- 🎥 Analyze YouTube videos and local media files
- 🎙️ Automatic speech-to-text transcription
- 📝 AI-generated meeting summaries
- 📌 Smart title generation
- ✅ Action item extraction
- 🎯 Key decision identification
- ❓ Open question extraction
- 💬 Chat with the transcript using Retrieval-Augmented Generation (RAG)
- 🎨 Modern Streamlit dashboard

---

## 🛠️ Tech Stack

**Frontend**

**Backend**
- Python

**AI & LLM**
- Mistral AI
- LangChain

**Audio Processing**
